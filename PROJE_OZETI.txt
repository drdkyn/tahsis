⚡ HIZLI BAŞLANGICI (5 dakika)
─────────────────────────────────────────────────────────────────────────────

1. Terminal'i aç ve şu komutları çalıştır:
   
   cd emeklilik-hesaplama-app
   npm install
   npm run dev

2. Tarayıcıda aç: http://localhost:3000

3. Form alanlarını doldur ve hesapla!

🌐 GITHUB'A YÜKLEME (10 dakika)
─────────────────────────────────────────────────────────────────────────────

1. GitHub.com'da yeni repository oluştur
   (Adı: emeklilik-hesaplama)

2. Terminal'de:
   
   git init
   git remote add origin https://github.com/USERNAME/emeklilik-hesaplama.git
   git branch -M main
   git add .
   git commit -m "Initial commit"
   git push -u origin main

3. Vercel.com'da otomatik deploy et!
